# Windows Server Admin Notes

Bu repo, Windows Server, Active Directory, Group Policy, File Server, Print Server ve temel sistem yönetimi süreçlerine yönelik teknik notları içermektedir.

## Amaç

Kurumsal bilgi işlem süreçlerinde kullanılan Windows Server yönetimi, kullanıcı politikaları, servis kontrolleri ve sistem dokümantasyonu çalışmalarını düzenli bir teknik arşiv haline getirmek.

## İçerik Planı

- Active Directory yönetim notları
- Group Policy yapılandırmaları
- Windows Server servis kontrolleri
- File Server yetki ve paylaşım notları
- Print Server sorun giderme notları
- BGInfo dağıtım notları
- Temel sistem güvenliği kontrolleri

## Güvenlik Notu

Bu repoda gerçek kurum adı, gerçek domain, gerçek IP adresi, kullanıcı adı veya kişisel veri paylaşılmamalıdır. Tüm örnekler test ortamına uygun ve anonim olacak şekilde hazırlanmalıdır.
